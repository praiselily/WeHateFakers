# WeHateFakers
Nothing is more annoying than getting bypassed by a faker. This script provides possible detection methods for faker bypasses and works on all operating systems (Windows, Linux, and macOS).

You can either import it using PowerShell or download the Java version from the Releases section.

PowerShell (Windows only)
`powershell -ExecutionPolicy Bypass -Command "iwr https://raw.githubusercontent.com/praiselily/WeHateFakers/refs/heads/main/HotspotLogs.ps1 | iex"

Java (Any other OS asw as Windows)
`cd "path/to/FakerFinder.java" && javac FakerFinder.java
`java HotspotDetector
